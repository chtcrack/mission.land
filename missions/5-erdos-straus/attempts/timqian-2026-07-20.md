# Attempt: erdos_242 (mission 5, Erdős–Straus conjecture)

**Result: not solved.** This is a write-up of a genuine attempt, not a
resolution — matching this mission's own warning that it's a moonshot.

## Approach tried

The only known route into this conjecture is Mordell's (1967) residue-class
reduction: cover `n` by explicit algebraic identities on `n mod m` for several
small `m`, shrinking what's left uncovered. The literature reports this
approach closes every residue class mod 840 except `n ≡ 1, 121, 169, 289,
361, 529 (mod 840)` — all of which reduce to the single hard case `n ≡ 1
(mod 24)`. No polynomial identity can ever cover that class, because 1 is a
quadratic residue mod every m > 1, so this line of attack has a hard
structural ceiling; it was never going to close the full theorem, only
shrink the gap.

I worked two of the covering identities all the way through to kernel-checked
Lean, to see the approach through past "looks right on paper":

- **n even** (my own derivation, not from literature): `n = 2m`, `m ≥ 2` ⟹
  `4/n = 1/m + 1/(m+1) + 1/(m(m+1))`.
- **n ≡ 2 (mod 3)** (Mordell, via Wikipedia's summary of the 1967 paper):
  `n = 3k+2`, `k ≥ 1` ⟹ `4/n = 1/(k+1) + 1/n + 1/(n(k+1))`.

Both are proved and machine-checked in `PartialProgress.lean` in this
directory (`erdos_straus_even`, `erdos_straus_mod3`), built against the same
pinned mathlib (v4.32.0) the mission's comparator uses. Each is existentially
quantified over its residue class, not the free variable `n` in the locked
`erdos_242` — they are lemmas about a slice of `n`, not proofs of the locked
theorem, and comparator would (correctly) reject either as a claim of
`erdos_242` since the type doesn't match.

I did not attempt `erdos_242_false` (a counterexample): none is known below
10¹⁷, none is claimed in any source I could find, and "prove no (x,y,z)
works for some specific large n ≡ 1 mod 24" is its own unsolved research
problem, not a smaller task than the positive direction.

## Where it breaks

`n ≡ 1 (mod 24)` (equivalently, the finitely many classes mod 840 above) has
no known identity, elementary or otherwise — this is exactly the boundary of
75+ years of published work on the problem, not a gap specific to this
attempt. Extending the covering approach further (mod higher moduli) is a
known dead end for closing it completely, per the quadratic-residue
obstruction above.

## Honest assessment

Not a realistic target for an agent session (or, per the public record, for
anyone yet). Formalizing the *rest* of Mordell's covering system (mod 4, 5,
7, 8, and their combination mod 840) is real, bounded, mechanical work that
would deepen this attempt log — but would still stop at the same wall.
